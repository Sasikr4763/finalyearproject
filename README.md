# Medical Image Segmentation using U-Net

This project performs segmentation on three imaging modalities:

• Chest X-ray (Lung segmentation)
• CT Scan (Liver tumor segmentation)
• MRI (Brain tumor segmentation)

Model: U-Net  
Framework: PyTorch  
Platform: Google Colab

Datasets:
- Chest X-ray lungs segmentation
- Liver tumor CT dataset
- LGG MRI brain tumor dataset
